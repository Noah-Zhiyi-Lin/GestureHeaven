示例动作图片说明：

1. 图片要求：
- 格式：jpg, png 或 jpeg
- 建议尺寸：800x600像素
- 背景：最好是纯色或简单背景
- 人物：清晰的全身姿势

2. 建议的动作类型：
- 基本站姿
- 举手
- 伸展
- 弯腰
- 单腿站立
- 深蹲
- 瑜伽基本姿势
- 简单舞蹈动作

3. 命名规则：
- 使用数字序号，如：pose1.jpg, pose2.jpg
- 或使用描述性名称，如：stand.jpg, raise_hands.jpg

4. 注意事项：
- 动作要清晰可见
- 避免过于复杂的动作
- 确保动作安全可模仿
- 姿势之间要有明显区别
